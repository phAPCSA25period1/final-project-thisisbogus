# Untitled Collaborative Writing CLI Program

UCWCLI is a program designed to break a writer out of a rut. It provides a collaborative writing game that can have an amount of players and rounds that you can decide.

## Usage

Input the amount of players and rounds you would like.
```java
//creates a game with three players and three rounds
How many players are there?:
3

How many rounds would you like?:
3
```
Then, input each player's name
```
#sets the first player's name as Jeff
What is Player 1's name?:
Jeff
```
The players can then each write segments of a story each round. Don't finish the story until it tells you to. There's a certain fun element, but that's a surprise!
